# 旧金山创业公司公开信息来源

## 一、最佳来源（直接链接）

### 1. Y Combinator 公司目录
**URL:** https://www.ycombinator.com/companies

- 筛选条件：Location = San Francisco, Status = Active
- 按行业筛选：AI, Developer Tools, SaaS, Fintech 等
- 信息包含：公司简介、官网、团队规模、融资阶段
- **优点**：高质量创业公司，通常在招人

### 2. Wellfound (原 AngelList Talent)
**URL:** https://wellfound.com/jobs

- 直接查看招聘职位
- 可以直接申请
- 公司页面有团队信息
- **优点**：有明确招聘需求

### 3. Built In San Francisco
**URL:** https://builtin.com/san-francisco

- 本地科技公司列表
- 按公司规模、行业筛选
- 有公司文化介绍
- **优点**：专注湾区

### 4. Crunchbase (免费版)
**URL:** https://www.crunchbase.com/discover/organizations

- 筛选：Headquarters = San Francisco, Founded > 2018
- 查看融资历史
- 找到最近融资的公司（可能在招人）
- **优点**：融资信息准确

### 5. LinkedIn Jobs
**URL:** https://www.linkedin.com/jobs/

- 搜索：[职位] + San Francisco + Startup
- 公司页面有员工信息
- 可以找到招聘经理
- **优点**：直接看到招聘意向

## 三、获取联系方式的合规方法

### 方法 1：公司官网
```
1. 访问公司官网
2. 查找 Careers / Jobs / About / Contact 页面
3. 记录：
   - 招聘邮箱（jobs@company.com, careers@）
   - 通用联系邮箱（hello@, contact@）
   - 团队页面的个人信息
```

### 方法 2：LinkedIn（不抓取）
```
1. 搜索公司
2. 查看 "People" 标签
3. 找到：Recruiter, HR, Hiring Manager, Founder/CEO
4. 记录姓名和职位
5. 通过官网推测邮箱格式
```

### 方法 3：GitHub
```
1. 搜索公司的 GitHub 组织
2. 查看组织页面的联系信息
3. 查看主要贡献者的 profile（可能有邮箱）
```

### 方法 4：新闻/博客
```
1. 搜索 "[公司名] TechCrunch/VentureBeat"
2. 新闻中常提到创始人/CEO
3. 公司博客可能有作者邮箱
```

## 四、公司评估标准

### 优先级评分（1-5）

| 因素 | 5分 | 3分 | 1分 |
|------|-----|-----|-----|
| 融资阶段 | A/B轮，刚融资 | Seed/天使 | Pre-seed/未融资 |
| 团队规模 | 20-100人 | 10-20人 | <10人 |
| 招聘状态 | 有公开职位 | 官网有招聘页 | 无招聘信息 |
| 行业匹配 | 完美匹配你背景 | 相关 | 不太相关 |
| 公司势头 | 最近有新闻/产品 | 稳定运营 | 沉寂 |

### "MicroHard 氛围"公司特征
- 扁平结构
- 远程/混合工作
- 技术驱动文化
- 透明沟通
- 重视工程质量

**查找方式：**
- Glassdoor 评价
- Blind 匿名评论
- LinkedIn 员工动态
- 公司博客/工程博客

## 六、数据收集工作流

```
每周花 2-3 小时：

周一：
  1. 查看 Y Combinator 目录，筛选新公司
  2. 检查 Wellfound 新职位
  3. 添加 10-15 家公司到列表

周三：
  1. 研究已添加公司的详细信息
  2. 查找联系方式
  3. 评估优先级

周五：
  1. 准备下周要发送的邮件
  2. 个性化每封邮件
  3. 安排发送时间
```

## 注意事项

1. **手动收集** — 不使用自动化抓取工具
2. **验证信息** — 确保邮箱有效再发送
3. **记录来源** — 记录每个联系方式的获取方式
4. **尊重隐私** — 不收集非公开信息
5. **保持更新** — 公司信息可能变化
