# 跟进策略指南

## 跟进时间线

```
Day 0   ──────►  发送初始邮件
                 │
Day 3-5 ──────►  跟进 #1（简短提醒）
                 │
Day 8-10 ─────►  跟进 #2（提供新价值）
                 │
Day 15-20 ────►  最终跟进（友好告别）
                 │
                 ▼
            移入冷名单（6个月后可再联系）
```

## 跟进规则

### 1. 频率限制
- **单个联系人：** 最多 4 封邮件（含初始邮件）
- **同一公司不同联系人：** 最多联系 2-3 人
- **两封邮件间隔：** 至少 3 个工作日

### 2. 每次跟进要添加价值
| 跟进次数 | 内容策略 |
|----------|----------|
| 第1次 | 简短提醒，可能添加一个新信息点 |
| 第2次 | 分享相关资源（文章、开源项目、行业见解） |
| 第3次 | 友好告别，留下联系方式 |

### 3. 停止跟进的信号
- 收件人明确表示不感兴趣
- 邮件被标记为垃圾邮件
- 邮箱退信
- 收件人要求停止联系

## 回复处理流程

### 积极回复
```
收到积极回复
      │
      ▼
24小时内回复
      │
      ▼
建议具体时间（给2-3个选项）
      │
      ▼
发送日历邀请
      │
      ▼
通话前1天发提醒
```

**回复模板：**
```
Hi {{name}},

Thanks for getting back to me!

I'm available:
- {{option_1}}
- {{option_2}}
- {{option_3}}

Let me know what works, and I'll send a calendar invite.

Looking forward to it!

Best,
{{your_name}}
```

### "暂时没有职位"回复
```
Hi {{name}},

Thanks for letting me know. Totally understand.

Would it be alright if I check back in a few months?
And if any roles open up before then, feel free to reach out.

Best,
{{your_name}}
```

### "请联系 XX"回复
```
Hi {{name}},

Thanks so much for the referral! I'll reach out to {{referred_person}}.

Really appreciate you taking the time.

Best,
{{your_name}}
```

### 无回复处理
1. 按时间线发送跟进
2. 所有跟进使用同一邮件线程（Re: 原主题）
3. 保持专业友好
4. 最终跟进后移入冷名单

## 跟进邮件撰写技巧

### 保持简短
- 跟进邮件应该比初始邮件更短
- 50-80 词即可
- 一个明确的 CTA

### 不要道歉
**不要说：**
- "Sorry to bother you again"
- "I know you're busy"
- "Apologies for following up"

**可以说：**
- "Following up on my note from Tuesday"
- "Wanted to circle back"
- "Quick follow-up"

### 添加新元素
每次跟进至少有一个新内容：
- 你完成的新项目
- 公司的最新新闻（"Saw your Series B announcement - congrats!"）
- 相关行业文章
- 新的技能或认证

## 追踪系统

### 状态标签
| 状态 | 含义 | 下一步 |
|------|------|--------|
| `new` | 刚添加 | 准备初始邮件 |
| `contacted` | 已发初始邮件 | 等待回复 |
| `followup_1` | 已发第1次跟进 | 等待/准备第2次 |
| `followup_2` | 已发第2次跟进 | 等待/准备最终 |
| `followup_final` | 已发最终跟进 | 移入冷名单 |
| `replied` | 收到回复 | 根据回复类型处理 |
| `scheduled` | 已安排通话 | 准备通话 |
| `not_interested` | 明确拒绝 | 存档 |
| `cold` | 冷名单 | 6个月后可再联系 |

### 每日检查清单
- [ ] 检查新回复
- [ ] 处理到期的跟进
- [ ] 更新联系人状态
- [ ] 准备明天的邮件

### 每周回顾
- 发送了多少封邮件？
- 回复率是多少？
- 哪种邮件效果最好？
- 需要调整策略吗？

## 指标追踪

### 关键指标
| 指标 | 目标 | 计算方式 |
|------|------|----------|
| 回复率 | >10% | 回复数 / 发送数 |
| 正面回复率 | >5% | 正面回复 / 发送数 |
| 面试转化率 | >2% | 面试数 / 发送数 |

### 红旗信号
- 回复率 < 5%：检查邮件质量
- 大量退信：验证邮箱有效性
- 被标记为垃圾邮件：停止发送，检查内容

---

## 时区注意事项

### 最佳发送时间（PST）
- **周二至周四**：最佳
- **周一**：可以，但早上可能被淹没
- **周五**：下午避免
- **周末**：不发送

### 最佳时段
- **9:00-11:00 AM**：首选
- **2:00-4:00 PM**：次选
- **避免**：12:00-1:00 PM（午餐）、下班后

### 如果你不在 PST 时区
使用邮件调度功能确保邮件在目标时区的最佳时间发送。
